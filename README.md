# 💰 Personal Expense Tracker

A complete single-page application (SPA) for tracking personal expenses with visual analytics. Built with vanilla JavaScript, Tailwind CSS, and Chart.js.

## ✨ Features

### Core Requirements
- ✅ Add new expense (Title, Amount, Category, Date, Notes)
- ✅ View all expenses sorted by date (most recent first)
- ✅ Edit existing expenses
- ✅ Delete expenses with confirmation
- ✅ Total spent summary
- ✅ Category breakdown pie chart

### Bonus Features
- ✅ Filter by date range (from/to)
- ✅ Filter by category dropdown
- ✅ Input validation (no negatives, required fields, no future dates)
- ✅ Responsive design (table on desktop, cards on mobile)
- ✅ Smooth UI transitions & error animations
- ✅ Tailwind CSS + Chart.js integration

## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html` in any modern browser
3. No server or installation required!

## 🧪 How to Test

| Action | Steps |
|--------|-------|
| Add Expense | Fill form → Click "Add Expense" |
| Edit Expense | Click orange edit icon → Modify → Click "Update" |
| Delete Expense | Click red trash icon → Confirm |
| Filter Expenses | Select date range/category → Click "Apply" |
| Mobile View | Resize browser or use device toolbar |

## 💾 Data Persistence

All expenses are automatically saved to your browser's localStorage. Refresh the page or close the browser - your data remains!
